# CRM Management System

A full-stack Customer Relationship Management (CRM) system built using the MERN stack. This application helps manage customer information efficiently through a user-friendly interface.

## 🚀 Features

- User Authentication
- Add Customers
- View Customer List
- Update Customer Details
- Delete Customers
- Responsive User Interface

## 🛠️ Technologies Used

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/smariammal-lab/crm-management-system.git
```

### Install Backend

```bash
npm install
```

### Install Frontend

```bash
cd crm-frontend
npm install
```

### Create Environment Variables

Create a `.env` file in the backend folder.

Example:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
cd crm-frontend
npm start
```
